# Diplom 2 — API-тесты для Stellar Burgers

## 📌 Описание
Автотесты для REST API проекта [Stellar Burgers](https://stellarburgers.nomoreparties.site/), написанные с использованием `pytest`, `requests`, `allure`.

## 🧪 Стек технологий
- Python 3.11
- Pytest
- Requests
- Faker (для генерации тестовых данных)
- Allure (для отчётов)
